## 🚀 Requisitos del sistema

- **PHP >= 8.1**
- **Composer**
- **MySQL o MariaDB**
- **Node.js y npm** (opcional, para compilar assets con Vite)
- **Laravel 10+**
- **Servidor local: Laragon, XAMPP, WAMP o similar**
